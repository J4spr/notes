
| bekijk parkeer geschiedenis | beschrijving                                                                                                                                               |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Doel                        | Beheerder wilt kijken naar de geschiedenis van alle reservaties en parkeerplekken                                                                          |
| Actoren                     | Primair: Beheerder<br>Secundair: parkeersysteem                                                                                                            |
| Precondities                | personeel moet een beheerder zijn                                                                                                                          |
| Postcondities               | Toon de hele geschiedenis                                                                                                                                  |
| Trigger event               | beheerder opent dashboard                                                                                                                                  |
| Hoofdsucces scenario        | 1. beheerder navigeert naar de parkeergeschiedenis pagina<br>2. het parkeersysteem laat de geschiedenis zien<br>3. beheerder kan filteren op alle tabellen |
| Alternatieve scenario's     | de medewerker is geen beheerder waardoor hij de geschiedenis niet kan zien.                                                                                |
