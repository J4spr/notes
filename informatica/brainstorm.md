
# min- en pluspunten

|          | +                           | -               |
| -------- | --------------------------- | --------------- |
| username | min aantal kararkters: 2    | spaties         |
|          | underscores en koppeltekens | geen leestekens |
| password | min aantal kararkters: 8    | spaties         |

 ---
 

# prompts 

1.  username: 
	- kies een username van minstens 2-24 characters
2.  password:
	- kies een paswoord van 8-24 characters

--- 
# responses

- username
	- username too long, choose another username between 2 and 24 characters
	- to short, try something longer
	- invalid character in the username, try something with valid characters (eg: - , _ )
	- username already exists
-  password
	- to short, try something longer
	- username too long, choose another username between 2 and 24 characters
	- to short, try something longer
	- invalid character in the username, try something with valid characters 
	 (eg: - , _ , . , / , ; , ! , ? )
